# Firefox快速开始指南

## 5分钟快速安装

### 第1步：打开Firefox调试页面
在地址栏输入：`about:debugging`

### 第2步：加载扩展
1. 点击"此 Firefox"按钮
2. 点击"临时载入附加组件..."
3. 选择项目文件夹中的 `manifest.json` 文件
4. 点击确认

### 第3步：开始使用
- 按 `Ctrl+T` 打开新标签页，查看你的书签
- 按 `Alt+B` (Mac: `Cmd+B`) 打开侧边栏
- 点击书签图标设置默认文件夹

## 系统要求

- **Firefox版本**: >= 60.0
- **操作系统**: Windows / macOS / Linux
- **网络连接**: 建议开启（用于favicon加载）

## 常见问题

### Q: 新标签页没有显示？
**A:** 确认扩展已启用，尝试刷新新标签页或重启Firefox。

### Q: 书签图标不显示？
**A:** 检查网络连接，favicon需要从Google API获取。可以刷新页面重试。

### Q: 快捷键不工作？
**A:** 检查其他扩展是否占用了相同的快捷键，可以在Firefox设置中查看和修改。

### Q: 如何卸载？
**A:** 打开 `about:addons`，找到TabMark，点击"..."菜单，选择"移除"。
